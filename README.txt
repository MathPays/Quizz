Pour lancer le jeu, placer le contenu du dossier Code (sans le dossier) dans la partie src d'un projet Java.
Lancer le fichier Main.java pour lancer le jeu.
Lancer TestJoueur.java, TestQuestion.java ou TestTheme.java pour tester les fonctions des classes du package.
